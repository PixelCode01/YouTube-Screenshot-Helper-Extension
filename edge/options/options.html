Download Settings

Hide the browser's download shelf when saving screenshots (when supported)

Prompt for download location instead of using default Downloads folder

Relative paths only - Files will be saved within your Downloads folder.

Examples:
- Screenshots -> Downloads/Screenshots/
- YouTube/Captures -> Downloads/YouTube/Captures/
- Media/2025 -> Downloads/Media/2025/

Note: Absolute paths like /home/user/Pictures/ or C:\Users\Name\Pictures\ are not supported due to browser security restrictions.
